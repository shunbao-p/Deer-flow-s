# 新闻查询技能

本技能用于查询今日全球最热门的新闻，提供全面、结构化的新闻摘要。

## 核心功能

1. **实时新闻搜索**：使用web_search工具搜索今日全球最热门的新闻
2. **多源信息提取**：从多个新闻来源获取信息，避免单一来源偏见
3. **结构化呈现**：将新闻组织成易读的格式，包含关键信息
4. **来源标注**：为每条新闻提供明确的来源链接，确保信息可追溯

## 工作流程

当用户询问新闻相关问题时，请遵循以下步骤：

### 1. 确定搜索关键词
根据用户的具体问题，确定合适的搜索关键词。如果用户询问“今日全球最热新闻”，使用类似以下的关键词：
- "今日全球热门新闻 2026年3月20日"
- "top news today worldwide"
- "breaking news today"
- "今日头条新闻"

如果用户询问特定类别新闻（如科技、财经、体育），添加相应限定词。

### 2. 执行新闻搜索
使用`web_search`工具搜索新闻，获取最新结果。建议搜索2-3次以获取不同来源：
- 第一次搜索：获取主流新闻源（如BBC, CNN, Reuters等）
- 第二次搜索：获取本地或特定区域新闻（如果相关）
- 第三次搜索：获取特定类别新闻（如果用户指定）

### 3. 提取和组织信息
从搜索结果中提取关键信息：
- **新闻标题**：每条新闻的核心标题
- **新闻来源**：发布新闻的媒体或网站
- **发布时间**：如果可用，显示新闻发布时间
- **关键内容**：新闻的摘要或关键点
- **新闻链接**：原始新闻的URL

### 4. 结构化呈现
将新闻组织成以下结构呈现给用户：

```
## 📰 今日全球热门新闻摘要
**更新时间**：[当前日期和时间]

### 🌍 国际头条
1. **[新闻标题]** (来源：[新闻来源])
   - 关键内容：[新闻摘要]
   - [citation:来源名称](新闻URL)

2. **[新闻标题]** (来源：[新闻来源])
   - 关键内容：[新闻摘要]
   - [citation:来源名称](新闻URL)

### 💼 财经新闻
1. **[新闻标题]** (来源：[新闻来源])
   - 关键内容：[新闻摘要]
   - [citation:来源名称](新闻URL)

### 🔬 科技新闻
...

### ⚽ 体育新闻
...

### 🎭 娱乐新闻
...

## 📊 新闻热点分析
[简要分析今日新闻的主要趋势或热点话题]

## 🔗 新闻来源汇总
- [新闻来源1](URL1) - [标题简述]
- [新闻来源2](URL2) - [标题简述]
- ...
```

### 5. 添加引用和来源
**重要**：必须为每条新闻添加引用。使用格式：
```
关键内容描述 [citation:来源名称](URL)
```

所有引用的完整列表必须在报告末尾的"来源"部分中列出，格式为：
```
## 来源
- [来源名称](URL) - 简要描述
```

## 最佳实践

1. **时效性优先**：优先选择今日或最近24小时内的新闻
2. **多样性**：从不同政治立场和地区的媒体获取新闻，提供平衡的视角
3. **准确性**：核实关键信息，避免传播未经证实的信息
4. **简洁性**：每条新闻摘要控制在2-3句话内
5. **可读性**：使用emoji和格式化提高可读性，但不要过度

## 特殊场景处理

### 用户指定特定话题
如果用户询问特定话题的新闻（如"AI领域最新新闻"），调整搜索关键词并聚焦该话题。

### 用户询问本地新闻
如果用户询问特定地区或国家的新闻，添加地区限定词搜索。

### 用户需要深度分析
如果用户需要新闻的深度分析，可以在提供新闻摘要后添加分析部分，包括：
- 事件背景
- 可能的影响
- 相关方反应
- 未来趋势预测

## 示例查询和响应

**用户查询**：今日全球最热新闻是什么？

**技能响应**：
1. 搜索"今日全球热门新闻 2026年3月20日"和"top news today worldwide"
2. 从BBC、CNN、Reuters等来源提取新闻
3. 按类别组织新闻
4. 提供结构化摘要和完整来源列表

## 注意事项

1. **避免偏见**：从多个来源获取新闻，避免单一视角
2. **事实核查**：对于有争议的新闻，注明多方观点
3. **敏感内容**：处理敏感话题时保持中立和尊重的语气
4. **版权尊重**：引用新闻时注明原始来源，不要复制大量原文

通过遵循这些指南，你将能够为用户提供全面、准确、结构化的新闻信息服务。
